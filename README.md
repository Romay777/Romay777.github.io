## 💝 О проекте

Этот сайт - особенный подарок на День Святого Валентина для самого дорогого человека в моей жизни. Он рассказывает нашу историю любви через интерактивную временную линию, наполненную особенными моментами и тёплыми воспоминаниями.

### ✨ Особенности

- 💫 Анимированное сердце со случайными романтическими фразами
- 🎨 Стильный дизайн в формате временной линии
- 📸 Эффект полароидных фотографий с воспоминаниями
- ⏱️ Счётчик дней любви
- 💖 Анимированные фоновые сердечки
- 🌟 Плавные анимации появления событий

## 🛠️ Технологии

- HTML5
- CSS3 (с анимациями и трансформациями)
- JavaScript (Vanilla)
- Google Fonts:
  - Meow Script
  - Montserrat Alternates
  - Sour Gummy

## 💕 Как использовать

1. Нажмите на пульсирующее сердце вверху страницы для случайных признаний в любви
2. Прокручивайте страницу, чтобы увидеть историю
3. Наведите на фотографии для эффекта увеличения
4. Наслаждайтесь каждым моментом истории

